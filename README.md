Financial Loan Analysis Dashboard
Project Overview
This project analyzes financial loan data to understand loan distribution trends, borrower risk profiles, and repayment behavior. The analysis combines SQL for data querying, Python (Jupyter Notebook) for Exploratory Data Analysis (EDA), and Power BI for interactive dashboard visualization. The goal is to generate insights that help financial institutions make data-driven lending and risk management decisions.
________________________________________
Project Objectives
•	Analyze loan distribution patterns across different years, states, and borrower profiles.
•	Perform SQL-based analysis to examine loan grades, revolving balances, and borrower verification status.
•	Conduct Exploratory Data Analysis (EDA) using Python in Jupyter Notebook to identify trends and relationships in the dataset.
•	Develop an interactive Power BI dashboard to visualize loan performance, borrower risk indicators, and repayment behavior.
•	Provide actionable insights that help financial institutions better understand borrower risk and loan repayment trends.
________________________________________
Key Insights
1. Year-wise Loan Distribution
Loan disbursement shows noticeable variation across years, helping identify growth patterns and changes in lending activity over time.
2. Credit Grade and Revolving Balance
Borrowers with certain credit grades tend to have higher revolving balances, indicating different levels of credit risk across borrower segments.
3. Verified vs Non-Verified Borrowers
Verified borrowers generally demonstrate more stable repayment behavior, while non-verified borrowers may show relatively higher risk patterns.
4. State-wise Loan Status
Loan distribution varies significantly across states, with some states contributing a larger share of total loan approvals and repayments.
5. Home Ownership and Repayment Behavior
Borrowers with home ownership tend to exhibit stronger repayment patterns, indicating a relationship between housing status and financial stability.

Exploratory Data Analysis (EDA)
EDA was performed using Python in Jupyter Notebook to explore patterns and relationships in the loan dataset.
Key EDA steps included:
•	Data cleaning and handling missing values
•	Analyzing loan distribution trends
•	Evaluating borrower credit grades and revolving balances
•	Comparing repayment behavior across verification status
•	Visualizing state-wise loan distribution and borrower characteristics
Libraries used:
•	Pandas – Data manipulation
•	Matplotlib / Seaborn – Data visualization
